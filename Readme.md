# Bitwarden Transfer Assist

**Ruh Roah Raggy** We've forgotten the master password to our bitwarden account. Like we can still access our account man, but 
we can't use their export tool without the password. We'll have to manually copy everything over and somehow format it into a file
that bitwarden can then import in a new account. 

## Dependencies
- From Pip:
  - None
- Python:
  - Python 3.10 and up...I think
  - tkinter (should be built-in)
  - json (should be built-in)
 
 ## Running
 1. Download BitwardenTransferAssist.py
 2. run it with python: `python BitwardenTransferAssist.py`
